Washington State Data Breach Analysis & Visualization

Name: Chase Chemero

Project Description

In an increasingly digital world, data privacy has become a paramount concern for citizens and legislators alike. Washington State requires organizations to report data breaches that affect its residents, creating a rich dataset for public analysis. This project visualizes those breach notifications to uncover whether data breaches are becoming more frequent, which industries are most vulnerable, and how long it typically takes for an organization to notify the public after a breach occurs.

Methodology

The dataset used is Washington State's public data breach notification records, sourced from the Attorney General's office. Each record includes the reporting organization, breach date, notification date, number of residents affected, type of information compromised, and industry sector.

Data was cleaned and parsed using Python (pandas), with dates normalized and missing values handled through imputation or exclusion depending on the field. Notification lag was computed as the difference between the breach date and the date residents were notified. Visualizations were built with D3.js and are fully client-side, requiring no backend.

Designs

  • Design 1: Temporal Dashboard
    Area chart of residents affected per month over time, with filtering and summary stats.
  • Design 2: Categorical Network
    Bipartite diagram linking organization type to information type stolen.
  • Design 3: Heatmap View
    Calendar heatmap of breach counts and notification lag by month and year.
  • Final: Washington Data Breach Explorer
    Interactive dashboard with cross-chart filtering by information type across a pie chart, area chart, and bar chart.
